# Juego: Adivina la palabra
## Descripción
- Juego que trata de adivinar una palabra en un tiempo limitado.
- Las letras de la palabra salen desordenadas y tienes que escribir la palabra que se esconde tras su desorden.
- Acumulas puntuaciones en base al número de palabras adivinadas

Este código es un código sencillo de práctica en clase para practicar ciertos conocimientos adquiridos.
## Lenguaje de programación
**Kotlin**
